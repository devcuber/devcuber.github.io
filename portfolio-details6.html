ODOO [Facturación Honduras]

Información del Proyecto

  • Categoría: ERP web
  • Cliente: Clientes de Grupo Innova
  • Fecha Proyecto: Noviembre, 2014
  • Proyecto URL: Condifencial
  • Tecnologías: Odoo, Python

Odoo es un ERP de Software libre hecho en Bélgica, cree un módulo para adaptar la facturación de ODOO a las necesidades de las empresas hondureñas, incluyendo el CAI fecha de expiración de CAI, rangos de nuemeros de facturación. Este módulo se instala a todos los clientes de Grupo Innova, empresa distribuidora de ODOO oficial en Honduras.
